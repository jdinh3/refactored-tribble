CSS

CSS gives us:

  • Background color
  • Height and Width of sections (really any element)
  • Font Family
  • Color of the font
  • Font Weight
  • Line height of the font

We access CSS:

  1. Add a link tag to your head
  2. Inside the link, add a href to your external stylesheet, href will be a relative path
  3. We need to add the appropriate class or id to an element
